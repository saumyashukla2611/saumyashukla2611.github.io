Live Demo for CSS Box Model

April 1, 2018

Box Model provides us the layout of a web page. A rectangular box represents every HTML element here. Words such as padding, border, margin sound naive and familiar but from a front-end developer's perspective they hold much more importance. Through this article, I am shadowing upon the use of these words as a property in the box model.
