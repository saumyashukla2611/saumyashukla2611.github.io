Exploring the Magick Package in R

February 22, 2018

Looking for image transformations and changes in R? Here is your package.

While reading about various packages in R, I came across this super fun package called magick. It was published in December 2017 with version 1.6 and is a very comprehensive open source image processing library. The package supports various file formats and handlings. Well, the use of this package depends upon the user but I am planning to implement this in my Rshiny application.

Installation

To install and load the package in R:

install.packages("magick", repos =
  "http://cran.us.r-project.org/bin/windows/contrib/3.3/magick_1.6.zip")
library(magick)

I have set a working directory where I have saved all the images I am going to use for this demo. You can easily set it inside R Studio: Session → Set Working Directory. After this, there is no need for giving an absolute path.

Loading Images

To load the image file:

tj <- image_read('tj.jpg')
image_info(tj)
##   format width height colorspace matte filesize
## 1   JPEG   480    360       sRGB FALSE    26673

You can use image_display to see the image but it's only available to preview the image in an X11 window.

There are many transformation options available with this package which makes your task easier — no need to use another tool to crop, modify, scale, or even change coloring in the image. I have taken image examples from my favorite childhood cartoon series, Recess.

Border Transformation

p1 <- image_read("tj.jpg")
print(p1)
##   format width height colorspace matte filesize
## 1   JPEG   480    360       sRGB FALSE    26673

image_border(image_background(tj, "orange"), "#FFA500", "10x10")

Well, I think TJ would look smarter with the border around the image. You can include any color with the color code, and it also provides flexibility to change border dimensions.

Text Annotation

How about some text in the image? I can generate my own memes in R:

image_annotate(tj, "WHY ME??", size = 40, color = "ORANGE",
  boxcolor = "BLACK", degrees = 40, location = "+22+20")

You can change the degrees up to which you want to rotate the text, and the positioning can be changed. +22 refers to the left side position and +20 from the top.

Blur Effect

Well, hey Spinelli, why so blurrrr?

p2 <- image_read("spinelli.jpg")
image_blur(p2, 10, 5)

Here, 10 represents radius and 5 represents sigma value.

Charcoal Effect

p3 <- image_read("recess_gus.jpg")
image_charcoal(p3)

There are many other options available — you can check them out from the link provided in the reference section.

Animations

We can use animations as well, since all functions in magick support working with layers. 

p4 <- image_read("Mikey.gif") %>%
  image_scale("300x") %>%
  image_quantize(256)
print(p4)
##   format width height colorspace matte filesize
## 1    GIF   300    220       sRGB  TRUE        0
## 2    GIF   300    220       sRGB  TRUE        0
## 3    GIF   300    221       sRGB  TRUE        0
## 4    GIF   300    220       sRGB  TRUE        0
## 5    GIF   300    223       sRGB  TRUE        0

image_flip(p4)

%>% denotes piping in R.

Image Morphing

You can also morph images:

p5 <- image_scale(image_read("vincent.jpg"), "x100")
p6 <- image_scale(image_read("gretchen.jpg"), "x100")
frame <- image_morph(c(p5, p6), frames = 10)
image_animate(frame)

Fun, right?
